# springbootvue3-
基于springbootvue3的项目,前端基本上只用了vue3和elementplus，具有一个商店项目的登录，查询，评论，添加购物车，下订单，修改密码，后台管理等基本功能。图片不显示是因为没有图片，自己换图片就行了。
